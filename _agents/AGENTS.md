(Auggie, Claude Code, Antigravity, Codex). Auggie, Antigravity
a Codex ho čítajú natívne; Claude Code ho číta cez symlink `CLAUDE.md → AGENTS.md`.

[`docs/ai-agents.md`](docs/ai-agents.md).
